What?
- This is a Chromium extension that monitors the SAP Launchpad in the background for incoming orders. It triggers a full-screen visual and audio alerts when new orders arrive to notify  staff. It also features a dashboard for tracking shift statistics. 

How to run it:
- Download or clone the project files.
- Open your browser and navigate to edge://extensions or chrome://extensions.
- Enable Developer mode.
- Select "Load unpacked" and choose the project folder.

Technologies used:
- JavaScript, HTML, CSS
- Chrome extension API
- IndexedDB for local media storage
- Web Speech API for Text-to-speech alerts
